Introduction to Web API - Air Quality

  In this assignment, I choose a web api "Air Quality" by Weatherbit. You have to specify the latitude and longitude when querying, and the api will return you with the the information that can help you assess the quality of the air in the specific area. In my case, I chose the AQI (Air Quality Index, used to describe the quality of the air) from the data.

  I choose this because it is similar to what we have done in projects like Color Field, where we mapped some meaningful data to display different but related elements. AQI index in itself has some advantages: it has a numeric range from 0 - 500, making it easy to map; it has corresponding color for each of its subrange, offering explicit mapping ideas; and it is higly related to our daily life, offering implicit mapping ideas. Such mapping can be applied to color and transparancy to make an expreienced website.
